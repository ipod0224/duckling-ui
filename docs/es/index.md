# Duckling

Una interfaz web moderna y fácil de usar para [Docling](https://github.com/docling-project/docling) (IBM), la potente librería de conversión de documentos.

![Captura de Duckling](../screenshot.png)

## Resumen

Duckling ofrece una interfaz intuitiva para convertir documentos con Docling. Convierte PDFs, documentos de Word, presentaciones e imágenes (con OCR) a formatos estructurados listos para flujos de IA.

## Inicio rápido

Consulta **Getting Started** para instalar y ejecutar Duckling en Docker o en desarrollo local.

## Estado de la traducción

La documentación en español está en progreso. Algunas páginas pueden contener contenido temporal o parcialmente traducido.
