# Getting Started

Welcome to Duckling! This section will help you get up and running quickly.

## Prerequisites

Before installing Duckling, ensure you have the following:

- **Python 3.10+** (3.13 recommended)
- **Node.js 18+**
- **npm or yarn**
- **Git**
